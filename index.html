Видеоуроки Уроки

HTML

Для начала нужно создать файл в папке под названием index.html

Позже нужно нажать сочетание клавиш Shift+! и выбрать первое

Далее нужно вписать название сайта в строку:

< title>Название< /title>

Чтобы поменять фон на сайте нужно в body написать:

style="background-color:"код цвета";

Для того что бы ввести текст нужно вписать:

< a>Текст< /a>

" либо "

< p>Текст< /p>

Для того чтобы сделать текст большим или маленьким нужно вписать:

< h1>Текст< /h1>

- самый большой текст

< h2>Текст< /h2>

- текст поменьше

< h3>Текст< /h3>

- средний текст

< h4>Текст< /h4>

- текст поменьше среднего

< h5>Текст< /h5>

- самый маленький текст

Что бы сделать текст по которуму переходить на другой сайт/траницу нужно вписать:

< a href = "сайт/страницу">

(можно ещё дописать target="blank" чтобы окно открывалось в новой вкладке)

Чтобы добавить фотографию нужно написать:

< img scr="путь к файлу">

(перед этим надо переместить фотографию в папку с сайтом)

Чтобы добавить видео нужно писать:

< video controls > < source src="путь к видео"> < /video >

(перед этим нужно перекинуть видео в папку с сайтом)

Чтобы добавить музыку надо написать:

< audio controls> < source src="путь к музыке"> < /audio>

Чтобы сделать кнопку надо прописать:

< button>То что вы хотите было написано в кнопке< /button>

Чтобы сделать поисковую строку нужно написать:

< input type="text" name="search" placeholder="Поиск">

Чтобы сделать кнопку скачки файла нужно написать:

< a href="имя файла" download>Скачать< /a>

Чтобы добавить карту на сайт, нужно перейти на сайт "ЯндексКарты", написать в поисне что вы хотите, нажать на три точки в правом верхнем углу, нажать поделиться, скопировать строку где написано виджет с картой и вставить в "index.html"

Чтобы сделать значок для сайтанужно перейти по ссылке"https://www.favicon-generator.org/", выбрать файл и нажать на кнопку "Creane favicon". После нужно скачать сжатый архив и перенести файлы в нем в папку где находится сайт. После этого нужно скопировать код ниже на сайте где мы скачивали сжаиый архив и вставить его перед названием сайта в "index.html".

CSS

Сначало нужно создать файл в папке " style.css "

Чтобы подключить css к html нужно написать:

"< link rel="stylesheet" href=" style.css ">" в "< head >< /head>"

Чтобы использовать css нужно прописать после кода:

" class="любой класс" " и в style.css прописать "класс которы вы прописали. {} "

В {} можно прописать очень много всего например:

"width="кол-во пикселей"

Так-же в {} можно прописать:

margin-left:"кол-во пикселей от левой границы";, margin-right:"кол-во пикселей от правой границы";,margin-top:"кол-во пикселей от верхней границы";, margin-bottom:"кол-во пикселей от нижней границы";

Так-же в {} можно прописать:

max-width:"Максимальная длинна";, max-height:"Максимальная ширина";, min-width:"Минимальная длинна";, min-height:"Минимальная ширина";,
